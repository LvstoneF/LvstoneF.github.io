SM3

  • SM3 消息摘要，可以用MD5作为对比理解，
  • sm3是国家密码杂凑算法标准， 由国家密码局2010年公布。
  • 输出值为杂凑值长度为256比特（8*32）与国际标准SHA-256等长，
  • sm3设计的安全性为128比特，与256比特椭圆曲线/sm2,sm4/sms4，AES-128安全性相同

算法过程

  1. 填充

    • 对消息长度小于2^64位进行运算，填充方法与SHA-256一样，假设消息长度位L，在消息的末尾添加一个1，在添加k个0，使得，1+k+L = 448 mod(521)然后再添加一个64bit,填充后的m1的长度为512的倍数

Posted on 2019-08-07

512 最长特殊序列

题设:

  • 给定两个字符串，你需要从这两个字符串中找出最长的特殊序列。最长特殊序列定义如下：该序列为某字符串独有的最长子序列（即不能是其他字符串的子序列）。

    子序列可以通过删去字符串中的某些字符实现，但不能改变剩余字符的相对顺序。空序列为所有字符串的子序列，任何字符串为其自身的子序列。

    输入为两个字符串，输出最长特殊序列的长度。如果不存在，则返回 -1。

    • 示例 :

      输入: “aba”, “cdc”
      输出: 3
      解析: 最长特殊序列可为 “aba” (或 “cdc”)
      说明:

    • 两个字符串长度均小于100。
      字符串中的字符仅含有 ‘a’~’z’。

思路

  • 解题思路：
    • 如果两个str1 ,str2 str1==str2, 长度内容一样，那么就返回-1
    • 如果两个的长度不一样str1.length()> str2.length()，那么就一定是返回str1.length()

SM4

Posted on 2019-08-05

SM4

  • SM4是我国无线局域网标准WAPI中所采用的分组密码标准，然后被商用密码标准采用，（SM是商密的意思，目前公布的商密由SM2，SM3。 预计在国内的敏感但非机密的应用领域会逐渐取代3DES,AES等国外分组密码标准，用于通信加密，数据加密等应用场合。

  • 密钥长度，分组长度为128，安全性相当于AES-128，不适用于需要长期保密的数据。

算法

  • SM4 是一个分组算法，分组长度为128，密钥长度128，加密算法和密钥扩展都用到了32轮非线性迭代结构。解密与加密算法结构相同，只是轮加密的顺序相反，解密轮密钥是加密轮密钥的逆序。
  1. 术语和定义：
    • Block Length 分组长度
    • Key Length 密钥比特位数
    • Key Expansion Algorithm 密钥扩展算法
    • Rounds 轮函数的迭代次数
    • Word 长度为32bite的组
    • S-box S盒，8bite输入， 8 bite输出的置换
  2. 密钥参数和常量
    • 加密密钥128bite, 表示为 $MK = (MK_0 …MK_3)$$MK$表示32bite的字
    • 轮密钥 $(rk_0 … .. rk_[31])$, 为32比特字，轮密钥由加密密钥生成。
  • $FK = (FK_0 … FK_3)$为系统参数，$CK = (CK_0 … CK_(31)$ 为固定参数，用于密钥扩展，

  1. 轮函数F

    • 若输入为$(X_0 X_1, X_2, X_3)$, 轮密钥为$rk$ 轮函数$F(X_0 X_1, X_2, X_3) =$ (X[0])^T((X[1]^X[2]^X[3]^rk)

  2. 合成置换T

    • 由非线性变换 t 和线性变化L 复合而成 $T(.) = L(t(.))$
    1. 非线性变换 t
      • 由四个Sbox并行构成
      • 输入$A = (a0, a1, a2, a3) 输出为 B= （b0, b1, b2, b3） $ 则 $t(A) = (Sbox(a0), Sbox(a1), Sbox(a2), Sbox(a3))$
    2. 线性变换L
      • C = L(B) = B^(B <<< 2)^(B<<<10)^(B <<< 18)^(B <<< 24)
