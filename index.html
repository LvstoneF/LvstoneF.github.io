祖冲之算法

1. 介绍：

  • 中国自主研制的六密码，主要运用于4G网络中的国际标准密码算法
  • 该算法包括：
    1. AUC（祖冲之算法）
    2. 128-EEA3 （加密算法）
    3. 128-EIA3 （完整性算法）

2. 算法描述

  1. 1 算法整体描述
  • 上层 16级线性反馈移位寄存器（LFSR）
    中层 比特重组（BR）
    下层 非线性函数（F）

  • 结构图

线性反馈移位寄存器（16LFSR）

非线性函数F

  • F 包括2个32比特单元变量$R_1 R_2$

  • F 的输入为3个32为比特字$X_0 … X_2$ ，输出为一个32比特字$W_0$

SM4

  • SM4是我国无线局域网标准WAPI中所采用的分组密码标准，然后被商用密码标准采用，（SM是商密的意思，目前公布的商密由SM2，SM3。 预计在国内的敏感但非机密的应用领域会逐渐取代3DES,AES等国外分组密码标准，用于通信加密，数据加密等应用场合。

  • 密钥长度，分组长度为128，安全性相当于AES-128，不适用于需要长期保密的数据。

算法

  • SM4 是一个分组算法，分组长度为128，密钥长度128，加密算法和密钥扩展都用到了32轮非线性迭代结构。解密与加密算法结构相同，只是轮加密的顺序相反，解密轮密钥是加密轮密钥的逆序。

  1. 术语和定义：

    • Block Length 分组长度
    • Key Length 密钥比特位数
    • Key Expansion Algorithm 密钥扩展算法
    • Rounds 轮函数的迭代次数
    • Word 长度为32bite的组
    • S-box S盒，8bite输入， 8 bite输出的置换

  2. 密钥参数和常量

    • 加密密钥128bite, 表示为 $MK = (MK_0 …MK_3)$$MK$表示32bite的字
    • 轮密钥 $(rk_0 … .. rk_[31])$, 为32比特字，轮密钥由加密密钥生成。
    • $FK = (FK_0 … FK_3)$为系统参数，$CK = (CK_0 … CK_(31)$ 为固定参数，用于密钥扩展，

  3. 轮函数F

    • 若输入为$(X_0 X_1, X_2, X_3)$, 轮密钥为$rk$ 轮函数$F(X_0 X_1, X_2, X_3) =$ (X[0])^T((X[1]^X[2]^X[3]^rk)

  4. 合成置换T

    • 由非线性变换 t 和线性变化L 复合而成 $T(.) = L(t(.))$
    1. 非线性变换 t
      • 由四个Sbox并行构成
      • 输入$A = (a0, a1, a2, a3) 输出为 B= （b0, b1, b2, b3） $ 则 $t(A) = (Sbox(a0), Sbox(a1), Sbox(a2), Sbox(a3))$
    2. 线性变换L
      • C = L(B) = B^(B <<< 2)^(B<<<10)^(B <<< 18)^(B <<< 24)

  5. 密钥扩展算法

SM2椭圆曲线公钥算法

  • 国家密码管理局 10-12-1
  • 对基于RSA算法的电子认证系统，密钥管理系统，应用系统进行升级改造
  • 在我国商用密码体系中SM2 被用来代替RSA算法

椭圆曲线与SM2的关系

  • SM2采用的椭圆曲线方程为$y^2 = x^3+a*x+b$, 在通过确定a, b 确定了唯一的标准曲线。
  • 在通过确定其他参数，p(x, y)基点， k(k倍点)，来将曲线映射为加密算法

椭圆曲线的原理

  • 在基点p(x, y)处切线，交双曲线与一点Q，在Q点做关于x轴的垂线交双曲线的另一边Q1，则Q1, p满足 Q1 = 2p,
  • 在做Q1 与p 的连线，交椭圆与一点R，做R关于X轴的垂线，交椭圆的另一边R1，则R1，p满足R1 = 3P,
  • 如此循环 ，就可以找点一个kP点，将k作为私钥，p作为公钥

SM2 的应用

  1. 签名，验签
  2. 加密，解密
  3. 密钥协商计算
  • SM2 的各个国家的标准不一样，所以互相不兼容

SM２的特点

前面速度快，验签速度慢，与RSA相反

加解密速度和验签速度相当

SM2支持多大的信息量，签名结果是多少字节？加密结果？

​ 原始数据可无限长，签名结果为64字节，

​ 加密支持128G， 结果增加96字节

SM２相关的算法

  • SM1， 加密强度 128位
  • SM3, 密码杂凑算法，杂凑值32字节，
  • SM4，对称加密算法，加密强度128位

ECC算法-椭圆曲线算法

前言

  • 基于离散对数设计的密码。
  • ECC14 == RSA1024位密钥加密强度，而且计算量更小，速度更快，
  • 运用于我国的二代身份证，比特币也将ECC作为加密算法

椭圆曲线阿贝尔群

  • 已知集合和运算(G,*)如果是群，必须要满足以下要求
    1. 封闭性：$\forall a b \in G, a*b\in G$
    2. 结合性：$\forall a, b,c\in G, (ab)*c = a(b*c)$
    3. 单位元：$\xi e \in G, \forall a\in G, 有 ea = ae = a$
    4. 逆元：$\forall a \in G, \xi b\in G, 使得ab = ba = e$
  • 阿贝尔群还需要在满足
    • $ab =ba$
  • 将椭圆定义到阿贝尔群内
    • 任取椭圆曲线上两点 P， Q （若P，Q重合，则做切线），做直线交于椭圆的第三点R1，过R1做平行与Y轴的直线交椭圆与点R
    • 定义P+Q=R，加法的和也在椭圆曲线上，并具备加法交换律，结合律

有限域$F_p(a,b)$上

  • $F_p $ 上有p个元素，p为质素

  • 基本运算都要加上$mod(p)$

  • $F_p$内满足交换律，结合律，分配律

  • 公式

    $x_3 \equiv \lambda ^2 - x_1 - x_2(mod(p))$

    $y_3 \equiv \lambda(x_1-x_3) - y_1$

    $\lambda = \frac{y_2-y_1}{x_2- x_1} p\neq Q$

    $\lambda = \frac{3x_1^2 + a_1}{2y_1} p= Q$

椭圆曲线加密

  • 椭圆曲线上的阶，存在最小的N，使得n P = 0, 则n为阶，如果n不存在，则为无限阶

  • K = kGk为私钥，K 为公钥

  • 用户A选n_a为密钥，$P_a = n_a G$作为公钥，B向A发送消息$P_m$,可选取一个随机数k$C _m = {kG, P_m + kP_a}$

  • A 解密

    $P_m + kP_a - n_akG = P_m +k(n_aG)- n_akG = P_m$
