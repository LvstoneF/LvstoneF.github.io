  • 算法过程

    • id 名称 实现过程 填充方法
      第一步 填充 报文长度 L % 512 不等于 448，就需要填充，使得满足之前的条件 第一位填 （1），后面填上 k g个（0），使得$（L+1+k）mod (512) == 448$， 消息长度变为 $N*512+448 == L$
      第二部 记录消息长度 用64位来填充前消息长度$ L= N512+448+64 = (N+1)512$
      第三步 装入标准幻数 $A = (01234567){16}, B = (89ABCDEF){16}, C = (FEDCBA98){16}, D = (76543210){16}$
      第四步 4轮运算 1. 将每一512 字节分为16个小组，一组64位（8）个字节。
