ECC算法-椭圆曲线算法

前言

  • 基于离散对数设计的密码。
  • ECC14 == RSA1024位密钥加密强度，而且计算量更小，速度更快，
  • 运用于我国的二代身份证，比特币也将ECC作为加密算法

椭圆曲线阿贝尔群

  • 已知集合和运算(G,*)如果是群，必须要满足以下要求
    1. 封闭性：$\forall a b \in G, a*b\in G$
    2. 结合性：$\forall a, b,c\in G, (ab)*c = a(b*c)$
    3. 单位元：$\xi e \in G, \forall a\in G, 有 ea = ae = a$
    4. 逆元：$\forall a \in G, \xi b\in G, 使得ab = ba = e$
  • 阿贝尔群还需要在满足
    • $ab =ba$
  • 将椭圆定义到阿贝尔群内
    • 任取椭圆曲线上两点 P， Q （若P，Q重合，则做切线），做直线交于椭圆的第三点R1，过R1做平行与Y轴的直线交椭圆与点R
    • 定义P+Q=R，加法的和也在椭圆曲线上，并具备加法交换律，结合律

有限域$F_p(a,b)$上

  • $F_p $ 上有p个元素，p为质素

  • 基本运算都要加上$mod(p)$

  • $F_p$内满足交换律，结合律，分配律

  • 公式

    $x_3 \equiv \lambda ^2 - x_1 - x_2(mod(p))$

    $y_3 \equiv \lambda(x_1-x_3) - y_1$

    $\lambda = \frac{y_2-y_1}{x_2- x_1} p\neq Q$

    $\lambda = \frac{3x_1^2 + a_1}{2y_1} p= Q$

椭圆曲线加密

  • 椭圆曲线上的阶，存在最小的N，使得n P = 0, 则n为阶，如果n不存在，则为无限阶

  • K = kGk为私钥，K 为公钥

  • 用户A选n_a为密钥，$P_a = n_a G$作为公钥，B向A发送消息$P_m$,可选取一个随机数k$C _m = {kG, P_m + kP_a}$

  • A 解密

    $P_m + kP_a - n_akG = P_m +k(n_aG)- n_akG = P_m$
