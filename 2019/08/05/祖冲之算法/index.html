祖冲之算法

1. 介绍：

  • 中国自主研制的六密码，主要运用于4G网络中的国际标准密码算法
  • 该算法包括：
    1. AUC（祖冲之算法）
    2. 128-EEA3 （加密算法）
    3. 128-EIA3 （完整性算法）

2. 算法描述

  1. 1 算法整体描述
  • 上层 16级线性反馈移位寄存器（LFSR）
    中层 比特重组（BR）
    下层 非线性函数（F）

  • 结构图

线性反馈移位寄存器（16LFSR）

  • LFSR 包括16个31比特寄存器 $s_0, … s_{15}$

  • LFSR 包括两种工作模式：初始化模式( init )和工作模式(work)

    1. 3.1 初始化模式

    • LFSR接收一个31比特字u，u 是由非线性函数F 的32比特输入W 舍弃最低为比特得到， 即$u = w >> 1$, 计算过程如下：

      • LFSRWithInitLisationMode(u)

        { $(1) v = 2^{15} s_{15} + 2^{17}s_{13} + 2^{21}s_{10}+2^{20}s_{4} +(1+2^8s_0)mod(2^{31} -1)$

        $(2) s_{16} = (v+u) mod(s^{31}-1)$

        $(3) 如果s_{16} = 0,则试 s_{16} = 2_{31} -1$

        $(4) (s_1, s_2, … s_{16}) —– (s_0 ….s_{15})$

        }

    1. 3.2 工作模式
      • 工作模式下 FLSR不接收任何输入
        -       
$(1) s_{16} = 2^{15} s_{15} + 2^{17}s_{13} + 2^{21}s_{10}+2^{20}s_{4} +(1+2^8)s_0mod(2^{31} -1)$

比特重组

  • 输入为LFSR寄存器单元变量，$s_0 …. s_{15}$

  • 输出为4个32比特字$X_0 … X_3$

  • 计算过程

非线性函数F

  • F 包括2个32比特单元变量$R_1 R_2$

  • F 的输入为3个32为比特字$X_0 … X_2$ ，输出为一个32比特字$W_0$

  • 计算过程

    • S 为32bit的S盒变换，L1， L2为32比特线性变换

      • $L_1(X) = X+(X <<<2) + (X <<< 10) + (X<<<18) + (X <<< 24)$

      • $L_2(X) = X + (X <<< 8) + (X <<< 14) + (X <<< 22) + (X <<, 30)$

密钥装入

  • 将初始密钥 k 盒初始向量 iv 分别扩展为16个31比特作为LFSR寄存器的初始状态

    • $K_0 || … || K_{15}$
    • $iv_0 || … || iv_{15}$

  • k ,iv 都为8-bit

  • 密钥装入过程

  • （1）D 为 240 比特的常量，可按如下方式分成 16 个 15 比特的字串：
    $D = d0 ‖d1‖……‖d15 其中:
    d0 = 1000100110101112，
    d1= 0100110101111002，
    d2 = 1100010011010112，
    d3 = 0010011010111102，
    d4= 1010111100010012，
    d5 = 0110101111000102，
    d6 = 1110001001101012，
    d7 = 0001001101011112，
    d8 = 1001101011110002，
    d9 = 0101111000100112，
    d10 = 1101011110001002，
    d11= 0011010111100012，
    d12 = 1011110001001102，
    d13 = 0111100010011012，
    d14 = 1111000100110102，
    d15 = 1000111101011002。$

  • （2）$对 0≤i≤15，有 si = ki‖di‖iv_i。$
