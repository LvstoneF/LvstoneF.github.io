SM2椭圆曲线公钥算法

  • 国家密码管理局 10-12-1
  • 对基于RSA算法的电子认证系统，密钥管理系统，应用系统进行升级改造
  • 在我国商用密码体系中SM2 被用来代替RSA算法

椭圆曲线与SM2的关系

  • SM2采用的椭圆曲线方程为$y^2 = x^3+a*x+b$, 在通过确定a, b 确定了唯一的标准曲线。
  • 在通过确定其他参数，p(x, y)基点， k(k倍点)，来将曲线映射为加密算法

椭圆曲线的原理

  • 在基点p(x, y)处切线，交双曲线与一点Q，在Q点做关于x轴的垂线交双曲线的另一边Q1，则Q1, p满足 Q1 = 2p,
  • 在做Q1 与p 的连线，交椭圆与一点R，做R关于X轴的垂线，交椭圆的另一边R1，则R1，p满足R1 = 3P,
  • 如此循环 ，就可以找点一个kP点，将k作为私钥，p作为公钥

SM2 的应用

  1. 签名，验签
  2. 加密，解密
  3. 密钥协商计算
  • SM2 的各个国家的标准不一样，所以互相不兼容

SM２的特点

前面速度快，验签速度慢，与RSA相反

加解密速度和验签速度相当

SM2支持多大的信息量，签名结果是多少字节？加密结果？

​ 原始数据可无限长，签名结果为64字节，

​ 加密支持128G， 结果增加96字节

SM２相关的算法

  • SM1， 加密强度 128位
  • SM3, 密码杂凑算法，杂凑值32字节，
  • SM4，对称加密算法，加密强度128位
